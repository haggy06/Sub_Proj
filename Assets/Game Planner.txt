게임 컨텐츠 기획서

게임 개요
제목 : 
장르 : 퍼즐, 플랫포머, 스테이지

게임 소개 : 온갖 장애물과 함정이 즐비해 있는 사원을 헤쳐 나아가는 게임이다. (레퍼런스 : 파이어보이 앤 워터걸 _ 포레스트 템플)

주요 시스템
기본 이동 : 기본적인 좌우 이동이 불가능하며 점프로만 이동할 수 있다.
점프 : 마우스 드래그로 방향과 세기를 조절한 후 마우스를 떼 점프한다. 중력 방향 쪽으로는 점프가 불가능하다. (레퍼런스 : 앵그리버드 - 조준)
벽에 충돌 : 벽에 부딪칠 시  충돌 반대 방향으로 튕겨나간다. 일부 장애물의 경우 예외가 있을 수 있다. (레퍼런스 : 점프킹 - 벽 충돌)
천장에 충돌 : 특이사항 없음.
착지 : 착지 시 바닥의 마찰력에 따라 관성을 받거나 움직일 수 있다. (레퍼런스 : 마리오 - 땅/얼음 지형)

게임 플로우
게임 시작 시) 게임 제목 밑으로 플레이 버튼, 옵션 버튼, 게임 종료 버튼이 있다. (레퍼런스 : 할로우 나이트 - 타이틀)
	게임 종료 버튼 클릭 시) 화면이 페이드된 후 게임이 꺼진다. (레퍼런스 : 할로우 나이트 - 게임 종료)
	옵션 버튼 클릭 시) 효과음, 배경음을 조절할 수 있는 바가 있다. (레퍼런스 : 파이어보이 앤 워터걸 - 설정)
	플레이 버튼 클릭 시 - 튜토리얼 클리어 X) 스테이지 선택 화면 대신 튜토리얼 스테이지로 즉시 넘어간다. (레퍼런스 : 파이어보이 앤 워터걸 - 게임 처음 시작)
	플레이 버튼 클릭 시 - 튜토리얼 클리어 O) 플랫포머 형식의 스테이지 선택 창으로 이동한다. 이전 스테이지를 깨야 다음 스테이지로 갈 수 있는 길이 열린다. (레퍼런스 별의 커비 _ Wii - 스테이지 선택 지역)
		스테이지 선택) 플레이어가 관문 앞에 서면 UI가 표시되며 상호작용 시 화면이 페이드아웃되며 스테이지를 로드한다. (레퍼런스 : 별의 커비 _ Wii - 스테이지 입장)
			스테이지 입장 시) 로드 완료 시 화면이 다시 페이드인되며 컨트롤이 가능해진다. (레퍼런스 : 없음)
				사망 시) 게임 오버 UI가 나타난다. 재도전 버튼과 메뉴(스테이지 선택)으로 돌아가기 버튼이 있다. (레퍼런스 : 파이어보이 앤 워터걸 - 게임 오버)
				클리어 시) 클리어 타임, 점프 횟수, 수집 요소 등을 통해 점수가 매겨진다. 재도전 버튼과 이어하기(스테이지 선택으로 이동) 버튼이 있다. (레퍼런스 : 컵헤드 - 스테이지 클리어)

게임 디자인
무대 : 오래된 석재 사원의 내부이다. 튜토리얼 스테이지는 사원의 입구 부분이며 스테이지 선택 씬 또한 사원 내부이다. (레퍼런스 : 컵헤드 - 우툴두툴 산마루)
인게임 테마 : 픽셀 형태의 게임이며 화면에 픽셀 락이 걸려 있어 회전, 이동 시에도 픽셀로만 보인다. (레퍼런스 : 데드셀 - 인게임 화면)
UI 테마 : 인게임과 달리 픽셀 형태가 아니다. 사원과 어울리는 석재 느낌이다. (레퍼런스 : 파이어보이 앤 워터걸 - UI)
색감 : 점프킹 정도의 색감이다. (레퍼런스 : 점프킹)
플레이어 디자인
	외형 : 인디아나 존 같은 갈색 코트 겸 탐험복을 입고 있다. 한 칸 크기의 SD 개릭터이다. (레퍼런스 : 스펠렁키 2 - 플레이어)
	기본 : 그냥 조금씩 흔들리는 정도의 애니메이션이다. (레퍼런스 : 할로우 나이트 - 정지)
	점프 준비 : 몸을 살짝 웅크리고 뛸 준비를 한다. 점프 각도에 따라 왼쪽/오른쪽을 바라본다. 
	점프 : 공중제비를 돌며 점프한다. 착지하거나 벽에 충돌할 때까지 돈다. (레퍼런스 : 산나비 - 로프 액션 _ 공중)
	벽 충돌 : 충격을 받고 뒤로 밀려나는 자세를 취한다. (레퍼런스 : 점프킹 - 벽 충돌)
	착지 : 잠깐 몸을 수그렸다 일으키며 착지한다. (레퍼런스 : 할로우 나이트 - 착지)
UI 디자인
	설정 : 효과음 조절, 배경음 조절 바가 가로로 배치되어 있으며 하단에 OK 버튼이 있다. (레퍼런스 : 파이어보이 앤 워터걸 - 설정)
	스테이지 내부 : 화면 상단에 플레이 시간을 나타내 주는 타이머가 있다. 목표 시간 초과 시 빨간 글씨가 된다. (레퍼런스 : 파이어보이 앤 워터걸 - 스테이지 UI)
	게임 오버) 재도전 버튼과 메뉴(스테이지 선택)으로 돌아가기 버튼이 있다. (레퍼런스 : 파이어보이 앤 워터걸 - 게임 오버)
	클리어) 클리어 타임, 점프 횟수, 수집 요소 등을 통해 점수가 매겨진다. 재도전 버튼과 이어하기(스테이지 선택으로 이동) 버튼이 있다. (레퍼런스 : 컵헤드 - 스테이지 클리어)

장애물 구상
사망 관련
	사원 컨셉
		물리 공격
			낙사 : 맵 밑에 깔려 있는 장애물. 장애물 보단 맵 탈출 방지 느낌이다.
			가시 : 맵 곳곳에 나 있는 정적인 장애물.
			발사기 : 주기적으로 화살을 쏘는 장애물.
			창 - 트랩 : 밟으면 잠시 후 튀어나오는 장애물.
			창 - 주기 : 주기적으로 바닥에서 튀어나오는 장애물.
			해머 - 트랩 : 특정 위치에 가면 내려찍거나 후려버리는 장애물.
			해머 - 주기 : 주기적으로 내려찍거나 회전하는 장애물.
			화염 방사기 : 주기적으로 화염을 뿜는 장애물.		
		화학 공격
			염산 - 웅덩이 : 맵에 고여 있는 액체 형태의 장애물.
			염산 - 배수구 : 벽에 붙어 있는 장애물. 주기적/지속적으로 염산을 쏟아낸다.
			용암 - 웅덩이 : 맵에 고여 있는 액체 형태의 장애물
			용암 - 배수구 : 벽에 붙어 있는 장애물. 주기적/지속적으로 용암을 쏟아낸다.

	식물 컨셉(목재를 포함한 대부분의 식물은 화염, 용암 등에 닿으면 불타 버린다)
		물리 공격
			가시덩굴 : 맵 곳곳에 자라 있는 정적인 장애물. 불규칙적으로 자라나 있을 수 있다.
			파리지옥 : 바닥에 숨어있는 덫 같은 장애물. 입이 닫힌 상태에선 발판으로 쓸 수 있다. (레퍼런스 : 데드셀 - 영락한 수목원's 빨간 꽃)
			포자꽃 : 벽에 붙어 있는 장애물. 주기적으로 일정 구역에 포자를 퍼뜨린다.
			고사리 : 벽에 자라나 있는 장애물. 닿으면 플레이어를 끌고 벽 속으로 들어가버린다. (레퍼런스 : 레인 월드 - 장대 식물)
